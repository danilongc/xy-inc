Common Services (REST)

Esta é uma API para exposição de serviços REST. Atualmente a API conta apenas com serviços de busca de CEPs e endereços fazendo Web Crawling na página dos Correios.

Serviços

Todos os serviços estão disponíveis para request via GET e retornam a resposta em formato JSON.

Busca de endereço por CEP
http://myserver:8080/common-services-web/rest/address/zipCode/38025490
Busca de endereço por logradouro
http://myserver:8080/common-services-web/rest/address/name/Visconde do Abaete, Uberaba
